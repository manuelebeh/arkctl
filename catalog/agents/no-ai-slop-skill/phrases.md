# Phrases to remove

## Throat-clearing openers

Cut and state the point:

- "Here's the thing:" / "Here's what/this/that/why…"
- "The uncomfortable truth is" / "The truth is,"
- "It turns out" / "The real X is"
- "Let me be clear" / "I'll say it again"
- "I'm going to be honest" / "Can we talk about"
- "Let's dive in" / "Without further ado"

## Emphasis crutches

- "Full stop." / "Period." / "Let that sink in."
- "This matters because" / "Make no mistake" / "Here's why that matters"

## Business jargon

| Avoid | Prefer |
|-------|--------|
| Navigate (challenges) | Handle, address |
| Unpack | Explain, examine |
| Lean into | Accept, use |
| Landscape (abstract) | Field, situation, stack |
| Game-changer | Name the concrete change |
| Double down | Commit, increase |
| Deep dive | Analysis / look at |
| Moving forward | Next |
| Circle back | Return to |
| Leverage (filler) | Use |

## Empty adverbs / softeners

Prefer deletion unless meaning changes: `really`, `just`, `literally`,
`genuinely`, `honestly`, `simply`, `actually`, `deeply`, `truly`,
`fundamentally`, `inherently`, `inevitably`, `interestingly`, `importantly`,
`crucially`.

Also cut: "At its core", "In today's X", "It's worth noting",
"At the end of the day", "When it comes to", "In a world where",
"The reality is".

## Meta-commentary

- "In this section, we'll…" / "As we'll see…" / "Let me walk you through…"
- "The rest of this essay…" / "I want to explore…"
- "Hint:" / "Plot twist:" / "X is a feature, not a bug"

## Chatbot artifacts

- "I hope this helps!" / "Let me know if you need anything else"
- "Great question!" / "You're absolutely right!"
- "Happy to help!" / offer-to-continue closers

## Vague declaratives

Replace with the specific claim:

- "The reasons are structural"
- "The implications are significant"
- "The stakes are high"
- "The consequences are real"
