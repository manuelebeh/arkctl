# Structures to avoid

## Binary contrasts / negative parallelisms

State Y. Drop the negation runway.

| Pattern | Fix |
|---------|-----|
| "It's not X, it's Y" / "not just X but Y" | State Y |
| "The answer isn't X. It's Y." | State Y |
| "Not because X. Because Y." | Because Y |
| Tailing fragments: "…, no guessing" | Full clause |

## Negative listing / staccato drama

- "Not A. Not B. C." → say C
- Stacked one-word punch lines → merge into normal sentences
- "Noun. That's it. That's the thing." → complete sentence

## Rhetorical setups

- "What if…?" as fake insight
- "Think about it:" / "Here's what I mean:"
- "Honestly?" / "Look," as theatrical openers
- Question immediately answered for effect

## False agency

Inanimate subjects doing human verbs. Name the actor.

| Avoid | Prefer |
|-------|--------|
| "the decision emerges" | "the team decided" |
| "the data tells us" | "the metrics show" / "we measured" |
| "the culture shifts" | "people changed how they work" |
| "a complaint becomes a fix" | "we fixed it after the report" |

## Narrator-from-a-distance

- "Nobody designed this." → put the reader in the scene
- "People tend to…" → "You often…" or name who
- "This happens because…" → lead with the cause

## Passive voice (when it hides actors)

Prefer an actor when ownership matters. Passive is fine for genuine unknown
actors or standard technical voice ("the request is rejected").

## Rhythm traps

- Default rule-of-three lists → use one or two items
- Every paragraph ending on a punchy one-liner → vary endings
- Em/en dashes → commas, periods, colons, parentheses
- Synonym cycling (same idea renamed each sentence) → repeat the clearest term
- False ranges ("from X to Y" when unrelated) → list topics

## Content inflation (from Wikipedia / humanizer)

- Significance puff: "pivotal moment", "broader landscape", "testament to"
- Promotional travel-brochure tone: nestled, breathtaking, vibrant heritage
- Vague attributions: "Experts believe", "Observers note" without sources
- Formulaic "Despite challenges… continues to thrive"
- Generic conclusions: "The future looks bright"
- Diff-anchored docs: describe what it does, not the changelog narrative
- Aphorism formulas: "X is the Y of Z" → concrete claim
